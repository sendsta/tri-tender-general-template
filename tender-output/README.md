# Tender Output

**⚠️ This folder is an OUTPUT DESTINATION only.**

---

## Important Notice

The AI will create **fresh, custom documents** in this folder based on your specific tender requirements.

**There are NO templates to edit here.**

Each document is built from scratch using:
- Your uploaded tender document
- Your company profile information
- Your brand voice and style

---

## What Gets Created Here

| File | Description | Typical Length |
|------|-------------|----------------|
| `01_cover_letter.html` | Executive cover letter | 2-3 pages |
| `02_company_profile.html` | Company overview and capabilities | 5-10 pages |
| `03_methodology_and_scope.html` | Technical approach and methodology | 8-15 pages |
| `04_pricing_schedule.html` | Pricing and cost breakdown | 3-5 pages |
| `05_compliance_checklist.html` | Requirement compliance matrix | Complete |
| `99_appendices.html` | Supporting documents index | As needed |

---

## Document Format

All documents are created as **HTML files** that:

✅ Preview directly in any web browser  
✅ Include print-friendly styling  
✅ Can be exported to PDF via browser print  
✅ Link to `brand.css` for consistent styling  
✅ Are fully customised to your tender  

---

## Previewing Documents

### In Browser
Simply double-click any `.html` file to open in your default browser.

### Export to PDF
1. Open the HTML file in your browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF"
4. Adjust margins if needed
5. Save

### In VS Code
Use the Live Preview extension for real-time preview.

---

## Workflow

```
1. Upload documents → tender-input/
2. AI analyses requirements
3. AI creates custom documents → HERE (tender-output/)
4. You review each section
5. AI makes requested changes
6. Final documents ready for submission
```

---

## ⚠️ Do NOT

❌ Manually create template files here expecting the AI to fill them  
❌ Edit generated documents during the workflow (changes may be overwritten)  
❌ Delete files during active document generation  

---

## After Completion

Once your tender response is complete:

1. Review all documents thoroughly
2. Export to PDF if required
3. Archive this folder for your records
4. Submit according to tender requirements

---

## File Naming

Files are numbered for logical document order:

```
01_ = Cover Letter (first document)
02_ = Company Profile
03_ = Methodology
04_ = Pricing
05_ = Compliance
99_ = Appendices (last document)
```

This ensures correct ordering when viewing or compiling.
