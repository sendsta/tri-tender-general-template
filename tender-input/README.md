# Tender Input

**Place your tender and company documents here.**

---

## What to Upload

### Tender Documents (Required)

| Document Type | Purpose |
|---------------|---------|
| RFP / RFQ / ToR | Main tender requirements document |
| Specifications | Technical specifications or scope of work |
| Pricing Template | BOQ or pricing schedule template (if provided) |
| Addenda | Any clarifications or amendments |
| Evaluation Criteria | How submissions will be scored |

### Company Documents (Required)

| Document Type | Purpose |
|---------------|---------|
| Company Profile | Overview of your company |
| Registration Certificate | CIPC or equivalent |
| Tax Clearance | Valid tax compliance certificate |
| B-BBEE Certificate | Broad-Based Black Economic Empowerment |
| VAT Registration | If VAT registered |

### Supporting Documents (Recommended)

| Document Type | Purpose |
|---------------|---------|
| Key Personnel CVs | Qualifications of team members |
| Reference Letters | From previous clients |
| Past Proposals | Examples of previous tender responses |
| Certificates | Professional registrations, ISO, etc. |
| Insurance Policies | PI, Public Liability coverage |
| Financial Statements | If required by tender |

---

## Supported File Formats

The AI can read:
- **PDF** (`.pdf`)
- **Word** (`.docx`, `.doc`)
- **Excel** (`.xlsx`, `.xls`)
- **Text** (`.txt`, `.md`)
- **Images** (`.png`, `.jpg`) - for scanned documents

---

## File Naming Convention (Recommended)

For easier organisation, consider naming files like:

```
TENDER_MainDocument.pdf
TENDER_Specifications.pdf
TENDER_PricingTemplate.xlsx
COMPANY_Profile.pdf
COMPANY_Registration.pdf
COMPANY_TaxClearance.pdf
COMPANY_BBBEE.pdf
CV_JohnSmith_ProjectManager.pdf
REF_ClientName_Letter.pdf
CERT_ISO9001.pdf
```

---

## After Upload

Once your documents are in this folder:

1. Start the AI workflow
2. The AI will confirm receipt of all files
3. It will analyse the tender requirements
4. Extract your company information
5. Guide you through creating the response

---

## Important Notes

⚠️ **Do not delete files from here** during the tender response process

⚠️ **Keep original filenames** if they contain important references

⚠️ **Ensure documents are readable** (not password-protected)

⚠️ **Check file sizes** - very large files may take longer to process
